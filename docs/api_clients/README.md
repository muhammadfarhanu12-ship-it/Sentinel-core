# Sentinel Core API clients (examples)

These are minimal reference clients for the versioned API namespace: `BASE_URL/api/v1/*`.

## Configuration

- `BASE_URL` (default `http://localhost:8000`)
- `API_KEY` for `x-api-key` (required for the scan endpoint)
- `BEARER_TOKEN` for dashboard endpoints (optional if demo mode is enabled)

## JavaScript

- File: `docs/api_clients/javascript/client.js`
- Run (Node 18+): `node docs/api_clients/javascript/client.js`

## Python

- File: `docs/api_clients/python/client.py`
- Run: `python docs/api_clients/python/client.py`
